## Acuo Travel Agent Operation is a cloud-based backend application designed to streamline group booking and travel management. 
It provides robust support for front-end publishing automation and facilitates efficient, reliable handling of travel bookings.

As a **Model-Driven Power App**, the application harnesses Microsoftâ€™s platform capabilities to help clients integrate **AI** and 
automation features. These tools enable travel businesses to **optimize** workflows, **enhance** operational efficiency, and 
deliver **exceptional**  service experiences.

## Authentication
All API requests use OAuth 2.0 for authentication. You must obtain an access token to interact with the API.

Token needs to be refreshed periodically to maintain access.

Ensure that your access token is kept secure and never exposed in client-side code.

## Endpoints

### 1. Package Management

#### Get Package Info

```http
GET /cps_get_package_info(cps_external_package_id=<EXTERNAL PACKAGE ID>)
Content-Type: application/json
```
