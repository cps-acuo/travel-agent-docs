## Purpose of Tour Guide Reports

Tour guide reports provide detailed information about travelers for tour guides to check who is attending different tours.
The report is per package and will list all travelers on that package and their details, such as name, contact information, and any special requests.

To access tour guide reports, navigate to the Reports section in the main menu and select **Tour Guide Reports**.
</Step>
<Step>
## Generating a Tour Guide Report

To create a comprehensive tour guide report:

1. Navigate to the Tour Guide Report section
2. Set the package to report on. 
3. Click on **View Report** to generate the report based on your criteria.

</Step>
