## Accessing Flight Reports

Flight reports provide comprehensive information about all flights booked across your travel packages. To access flight reports, navigate to the Reports section in the main menu and select **Flight Reports**.

These reports help travel agents track all flight bookings, monitor schedule changes, and ensure travelers have up-to-date flight information.
</Step>
<Step>
## Generating a Flight Report

To generate a new flight report:

1. Navigate to the Flight Report section
2. Set the filter parameters (date range, destinations, airlines, etc.)
3. Click on **Generate Report** to create a new report based on your criteria

The system will compile all relevant flight information matching your criteria into a comprehensive report.
</Step>
<Step>
## Analyzing and Exporting Flight Data

Once your flight report is generated, you can:

- Sort and filter the data by various parameters (airline, date, destination, etc.)
- View passenger manifests for each flight
- Identify potential scheduling issues or conflicts
- Export the report in various formats (PDF, Excel, CSV) for sharing or further analysis
- Send notifications to travelers about flight details or changes
</Step>
