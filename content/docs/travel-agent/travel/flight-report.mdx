## Purpose of Flight Reports

Flight reports provide information to send to airline for listing of passages for a flight.

</Step>
<Step>
## Generating a Flight Report

To generate a new flight report:

1. Navigate to the Flight Report section.
2. Set the Flight with the package to report.
3. Click on **View** to create a new report based on your criteria.


</Step>
