Postal codes in Travel Agent are used to manage geographic locations. While it is uncommon to create new postal codes, this page provides an overview of existing postal codes and their structure.

The main fields on this page are:
- **Zip**: The postal code itself.
- **City**: The city associated with the postal code.
- **Country/Region Code**: The country or region for the postal code.
