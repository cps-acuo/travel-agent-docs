<Steps>
<Step>
## Navigating to the Items Page

The main **Items** page provides an overview of all existing items within the system. From here, you can view, manage, and create new items.

![The Items page in Travel Agent](/images/travel-agent/setup-items-view.png)
</Step>

<Step>
## Creating a New Item

To create a new item, click on the **New** button in the command bar. This will open the form for a new item.
</Step>

<Step>
## Defining the Item Details

In the **New Item** window, you need to fill in the details for the item. Once you have entered the required information, you can save the new item.

![The New Item creation window](/images/travel-agent/setup-items-form.png)
</Step>
</Steps>
