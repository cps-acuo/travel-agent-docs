<Steps>
<Step>
## Navigating to the Payment Methods Page

The main **Payment Methods** page provides an overview of all existing payment methods within the system. From here, you can view, manage, and create new payment methods.

![The Payment Methods page in Travel Agent](/images/travel-agent/setup-payment-method-view.png)
</Step>

<Step>
## Payment Method Types

When creating a new payment method, you will be prompted to select a type. The available types are:
- **Straumur**
- **Bank Account**
- **Credit Card**
- **PayPal**
- **Other**

![The Payment Method Types](/images/travel-agent/setup-payment-method-types.png)
</Step>

<Step>
## Defining the Payment Method Details

In the **New Payment Method** window, you need to fill in the details for the payment method. Once you have entered the required information, you can save the new payment method.

![The New Payment Method creation window](/images/travel-agent/setup-payment-method-new-form.png)
</Step>
</Steps>
