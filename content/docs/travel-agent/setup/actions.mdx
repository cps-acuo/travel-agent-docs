## Navigating to the Actions Page

The main **Actions** page provides an overview of all existing actions within the system. From here, you can view, manage, and create new actions.

![The Actions page in Travel Agent](/images/travel-agent/setup-actions-view.png)
</Step>


<Step>
## Creating a New Action

To create a new action, click on the **New** button in the command bar.

![The New Action creation window](/images/travel-agent/setup-new-action.png)


The main fields are **Action Name** and **Action Type**. For the **Action Name**, choose a descriptive name that clearly identifies the action (e.g., 'British Airways Flight'). You must also choose an **Action Type** from the dropdown menu. The options are as follows:
- Flight
- Hotel
- Tours
- Transport
- Food & Beverage
- Other



</Step>
