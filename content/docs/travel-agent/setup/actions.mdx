# Actions Setup


This guide explains how to set up and manage Actions in the Travel Agent system. Actions are used to automate various tasks and processes.

<Steps>
<Step>
## Navigating to the Actions Page

The main **Actions** page provides an overview of all existing actions within the system. From here, you can view, manage, and create new actions.

![The Actions page in Travel Agent](/images/travel-agent/setup-actions-view.png)
</Step>

<Step>
## Creating a New Action

To create a new action, click on the **New** button in the command bar. This will open the view for all actions.

</Step>

<Step>
## Defining the Action Details

In the **New Action** window, you need to fill in the details for the action. This includes providing a unique code and a descriptive name for the action. Once you have entered the required information, you can save the new action.

![The New Action creation window](/images/travel-agent/setup-new-action.png)
</Step>
</Steps>
