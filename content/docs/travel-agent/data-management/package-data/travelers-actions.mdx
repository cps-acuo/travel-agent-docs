title: Travelers Actions
description:  Overview of all actions for travelers booked.
icon: User
---

On this page users can see all actions for travelers on a Bookings. On this list Multi-select to make changes to different travelers actions.  
If a traveler has no actions, a new action can be created for the traveler from the Travel Package Actions form.

It is not possible to create a new action for a traveler that has no booking. The user must first create a booking for the traveler and then create an action for the traveler.
