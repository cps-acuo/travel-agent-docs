---
title: "Installation"
description: "Follow these steps to install the Acuo Travel Agent WordPress plugin on your site."
---




## 1. Download
   - Download the `travel-agent.zip` plugin package from the official source or repository provided by Acuo.

## 2. Upload
   - In your WordPress dashboard, navigate to `Plugins → Add New → Upload Plugin`. Select the `travel-agent.zip` file and click **Install Now** to upload the plugin.

## 3. Activate
   - After installation is complete, click **Activate** to enable the Acuo Travel Agent plugin on your site.
