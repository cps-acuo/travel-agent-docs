---
title: "Installation"
description: "Follow these steps to install the Acuo Travel Agent WordPress plugin on your site."
---



## 1. Download
   - Obtain the `travel-agent.zip` plugin package from the official source or repository.
## 2. Upload
   - In your WordPress admin, go to `Plugins → Add New → Upload Plugin`, select `travel-agent.zip`, then click **Install Now**.
## 3. Activate
   - Once installed, click **Activate** to enable the plugin.
