Navigate to `Settings → Travel Agent` in your WordPress dashboard to configure the following sections.

### 2.1 Expandable Iframe

Configure these options for the expandable iframe.

#### 🔒 Required

>- **Default Iframe URL**  
    Set this option to the URL where your [Booking Form](/docs/ta-bookingform) is hosted.

### 2.2 API & Filter

Configure these options for the API and filter settings.

#### 🔒 Required

>- **API Key**  
  Generate a key for the [Travel Agent Operations](/docs/travel-agent) platform in `API Key` field under the relevant brand. 
  ⚠️Store it securely—shown only once.  
>- **Package Post Type**  
  Custom post type for travel packages (e.g. `travel_package`).
>- **Destination Taxonomy**  
  Taxonomy for package destinations (e.g. `destination`).
>- **Experience Taxonomy**  
  Taxonomy for experience categories (e.g. `experience`).
>- **Period Taxonomy**  
  Taxonomy for time periods (e.g. `period`).
>- **Integration Status**  
  Switch between:
  >>- `Migration`: Disable new post creation (for data migration).
  >>- `Live`: Enable real-time post creation.

## 3. Save Settings

Click **Save Changes** at the bottom of the page to apply your configuration.
