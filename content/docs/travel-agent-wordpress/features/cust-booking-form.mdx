description: Customize the booking form in the Travel Agent WordPress plugin.
---


The booking form is fully customizable, allowing you to align its appearance and functionality with your site's design and branding requirements. Key customization options include:


## Key Features

- **Colors & Style**  
  - Configure background color, text color, and button styles to ensure cohesive branding throughout the booking experience.



## Configuration

All configuration options for the customizable booking form are available in **Settings → Travel Agent → Booking Form** within your WordPress admin dashboard. For detailed instructions on each setting, please refer to the **Setup** documentation.
