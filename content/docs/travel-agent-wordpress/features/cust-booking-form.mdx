The booking form can be customized to match your site’s design and branding. Key customization options include:

## Key Capabilities

- **Colors & Style**  
  - Background color, text color, and button styles for cohesive branding.


## Configuration

*Note:* All configuration options are available in **Settings → Travel Agent → Booking Form**.
Refer to the **Setup** documentation for detailed instructions on each setting.
