## 1. Automatic Package Synchronization

- **Seamless Synchronization**  
  New or updated travel packages in the [Power APP platform](/docs/travel-agent) are automatically synchronized with your WordPress site.
- **No Manual Intervention**  
  All changes made in the Power APP platform are reflected instantly on your website, eliminating the need for manual updates.


## 2. Automatic Post Creation

- **Dynamic Post Generation**  
  Each new or updated package automatically generates a corresponding WordPress post, utilizing your selected template or a default layout.
- **Comprehensive Content**  
  Generated posts include the package title, excerpt, images, and a fully configured booking button.


## 3. Customizable Booking Button

- **Flexible Styling**  
  Tailor the booking button’s text, colors, icons, and size to align with your site’s branding.
- **Expandable Iframe**  
  Clicking the button reveals an embedded booking form (`iframe`) on the same page.
- **Direct Bookings**  
  Users can book without leaving your site via the [booking form](/docs/travel-agent-web-bookings).

## 4. Customizable Booking Form

- **Brandable Design**  
  Personalize form fields, button styles, borders, and color schemes to match your brand identity.
- **Adaptive Layout**  
  Fine-tune corner radius, card backgrounds, and input focus states for a polished user experience.


## 5. Developer API

- **RESTful Endpoints**  
  Programmatically manage packages and bookings via JSON API.
- **Full CRUD Support**  
  Create, read, update, and delete travel packages from external applications.


## 6. Filter Grid

- **Advanced Filtering**  
  Enable visitors to filter travel packages by multiple criteria, including:
  - **Destination**
  - **Experience**
  - **Travel Period**
- **Enhanced Discovery**  
  Facilitate quick and accurate package selection with a robust multi-criteria search interface.
