## Overview

The Automatic Package Sync feature keeps your WordPress site in sync with the backend (e.g., [Travel Agent Operations](/docs/travel-agent)). When packages are added, updated, or deleted in the backend, the plugin automatically updates your WordPress site. This runs in the background with no manual steps needed.


## Travel Packages Table

The plugin includes a table within the WordPress admin area, displaying all available travel packages. This table presents key details such as package name, description, price, and availability. Administrators can efficiently view, edit, or delete packages, as well as perform actions on individual packages—such as refreshing post data, creating new posts, or removing outdated entries—directly from the table interface.


## Key Benefits

- **Real-Time Updates:** Packages sync in real time so your site always shows current offerings.
- **Automated Sync:** No manual updates needed — saves time and reduces errors.
- **Up-to-Date Content:** Visitors always see the latest package information.
- **Centralized Management:** Manage all packages from one place in WordPress.
- **Quick Actions:** Refresh data, create posts, or remove packages directly from the admin interface.
- **Booking Integration:** Works with the [Booking Form](/docs/travel-agent-web-bookings) so users can book directly from your site.
- **Scalable:** Handles growing numbers of packages without affecting performance.
- **Secure:** All data sync is handled securely.
- **Easy to Use:** Manage packages without technical expertise.
- **Mobile-Friendly:** Package pages and booking forms work on all devices.
