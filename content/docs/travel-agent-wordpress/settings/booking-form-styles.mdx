---
title: "Booking Form Styles"
description: "Learn about the key styling options for the booking form in the Travel Agent WordPress plugin."
---

Customize the appearance of the booking form fields and buttons.

- **Remove Button Color**  
  Color for the “Remove” button.
- **Corner Radius**  
  Border-radius for form elements (e.g. `4px`).
- **Border**  
  Color for the form container border.
- **Input Border**  
  Color for input field borders.
- **Form Card Background**  
  Background for traveler card sections.
- **Background Card**  
  Page background behind card sections.
- **Text Color**  
  Primary text color for form labels and inputs.
- **Muted Text Color**  
  Color for secondary/helper text.
- **Hover Background**  
  Background color on hover for selects.
- **Submit Button Color**  
  Background color for the submit button.
- **Submit Hover Color**  
  Background color on submit button hover.
