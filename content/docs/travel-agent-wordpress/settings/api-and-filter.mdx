---
title: "Api and Filter Settings"
description: "Learn about the key API and filter settings of the Travel Agent WordPress plugin."
---

Define custom post types, taxonomies, and API keys for external integrations.

- **API Key**  
  Generate an API key for external clients. Ensure this key is stored securely, as it will only be displayed once for security purposes.
- **Package Post Type**  
  Specify the custom post type used for travel packages (e.g., `travel_package`).
- **Destination Taxonomy**  
  Define the taxonomy used for package destinations (e.g., `destination`).
- **Experience Taxonomy**  
  Define the taxonomy used for experience categories (e.g., `experience`).
- **Period Taxonomy**  
  Define the taxonomy used for time periods (e.g., `period`).
- **Integration Status**  
  Switch between:
  - `Migration`: Disable automatic new post creation (for data migration).
  - `Live`: Enable real-time post creation.
- **Post Status**  
  Set the minimum post status required for display on the frontend (default: `published`).
