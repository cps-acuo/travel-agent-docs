Define custom post types, taxonomies, and API keys for external integrations.

- **API Key**  
  Generate a key for external clients. Store it securelyâ€”shown only once.  
- **Package Post Type**  
  Custom post type for travel packages (e.g. `travel_package`).
- **Destination Taxonomy**  
  Taxonomy for package destinations (e.g. `destination`).
- **Experience Taxonomy**  
  Taxonomy for experience categories (e.g. `experience`).
- **Period Taxonomy**  
  Taxonomy for time periods (e.g. `period`).
- **Integration Status**  
  Switch between:
  - `Migration`: Disable automatic new post creation (for data migration).
  - `Live`: Enable real-time post creation.
- **Post Status**  
  Minimum status to display on the frontend (default: `published`).
