---
title: "Expandable Iframe Settings"
description: "Learn about the key settings for the expandable iframe in the Travel Agent WordPress plugin."
---

Configure how the booking iframe appears on your travel package pages.

>- **Default Iframe URL**  
  URL for the booking iframe on your package posts.  
  _Example:_ `/docs/ta-bookingform`
>- **Button Text (Booking)**  
  Text displayed on the booking button (e.g. “Book Now”).
>- **Button Text (Waitlist)**  
  Text when a package is full (e.g. “Join Waitlist”).
>- **Close Button Text**  
  Text for closing the expanded iframe (e.g. “Close”).
>- **Button Color**  
  Hex or named color for the button background.
>- **Button Text Color**  
  Color for the button label.
>- **Button Width**  
  CSS width (e.g. `200px`, `50%`).
>- **Button Icon**  
  Icon class or URL for the booking button.
>- **Use URL Segment**  
  Toggle to infer the package via URL segment.
>- **URL Segment**  
  URL path segment matching the package external ID.
>- **Booking Redirect Link**  
  URL to send users after successful booking.
>- **Waitlist Redirect Link**  
  URL to send users after waitlist registration.
>- **Iframe Width**  
  Width of the booking iframe (e.g. `600px`).
>- **Iframe Height**  
  Height of the booking iframe (e.g. `800px`).
>- **Dynamic Resize**  
  Enable automatic height adjustment (requires TA booking page).
