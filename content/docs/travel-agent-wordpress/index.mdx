## **Overview** 

The Acuo Travel Agent WordPress Plugin connects your WordPress site to backend booking modules, like Acuo's Travel Agent Operation and Web Booking bookings. It lets you display travel packages, connect to bookings, and customize using Elementor along with WordPress taxonomies.

## What You Can Do

**Display Travel Packages**
Search and filter travel packages based on taxonomy or other properties assigned to package items. Automatically sync packages from backend.


## **Key Features**


### Acuo Taxonomy Search
This component introduces intelligent taxonomy-based search functionality, allowing users to filter posts and package listings according to rich metadata provided by Acuo’s backend.

### Acuo Item Package list 
This component introduces functionality, allowing users to view posts/package for an Item (a grouping of packages) from Acuo’s backend to see things like **Upcoming Departures** where you have similar packages with different departures and no need a new page for each package.

### Acuo Expandable Iframe 
An embeddable iframe component that integrates Acuo Travel Agent’s booking engine directly within WordPress. It allows designers to:
- Embed booking forms seamlessly.
- Control layout and responsive behavior.
- Provide a smooth user experience from browsing to booking.

### Booking and Package API Integration
A robust API service synchronizes package information and availability data between the backend system and the WordPress site:
- Automatically creates and updates a table of available offerings.
- Links each offering to a corresponding WordPress post with customizable layout.
- Dynamically updates taxonomy metadata and post status based on backend availability.
- Provides real-time PAX (passenger) availability, including seat counts per package.
- Automatically change to waitlist if no seats are available.

### Customization & Control
The plugin is tightly integrated with Elementor, giving designers control over the presentation layer. Developers can leverage custom components while still maintaining a flexible, WordPress-native development workflow. 

### Who Should Use This Plugin?
- Travel agencies seeking to showcase dynamic, up-to-date vacation packages.
- Booking platforms requiring real-time availability and automated updates.
- Designers and developers in need of a flexible, professional-grade plugin for custom travel solutions.
