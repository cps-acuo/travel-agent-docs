# Overview

**Acuo Travel Agent** is a cloud-based platform built for managing group bookings for tours and events. It integrates with both WordPress and Node.js web applications, helping tour operators and event organizers manage their operations and deliver great booking experiences.

## Modular Product Architecture

Acuo Travel Agent is made up of three modules. Each works independently and can be combined as needed:

1. **Operations:**
   A full-featured backend for managing package bookings and related activities. It includes tools for handling bookings, coordinating workflows, organizing booking data, and supporting day-to-day operational tasks in one place.

2. **Web Bookings:**
   A SaaS website (Node.js-based) that provides visitors with an easy-to-use booking interface for completing package bookings online. Booking results are sent as JSON to the backend for processing.

3. **WordPress Plugin:**
   A WordPress plugin for supporting and managing package content on your website. It enables taxonomy-based search, booking integration, and real-time data updates from the backend.
