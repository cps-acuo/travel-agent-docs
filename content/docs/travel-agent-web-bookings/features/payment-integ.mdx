There is a payment integration feature available in the Travel Agent Booking form product. This feature allows you to securely process payments through various payment gateways, ensuring a smooth and reliable transaction experience for your users.
# Payment Gateways
The Travel Agent Booking form supports integration with payment gateways, for creating payment session in the form and forwarding clients to a secure payment site, by 
by making a web request to a backend service.   Currently fully support by the Acuo Travel Agent Operations. 

The payments options in Acuo Travel Agent Operations is based on email links that have been created in the backend with a link included.
For more information on setting up and configuring payment integration, contact your system administrator.
